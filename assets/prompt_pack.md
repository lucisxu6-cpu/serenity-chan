# Prompt Pack

## A 股主题扫描

```text
使用 serenity-chan-stock-skill 深度扫描 A 股 [主题]。
请先做数据源计划，明确 A 股公告/财报/行情源，不要用美股数据源。
先排产业链瓶颈层级，再从至少 20 个候选中筛 3-5 个长线高胜率研究对象。
必须输出产业链地图、瓶颈层级排序、候选公司池、热门但降级方向。
每个对象必须说明：卡住什么环节、证据等级、财务兑现、估值赔率、缠论位置、证伪条件。
如果当前数据不足，请明确评级上限。
```

## 美股单股研究

```text
使用 serenity-chan-stock-skill 分析 [TICKER]。
请走 US 数据路由：SEC EDGAR/公司 IR 为财报和 filing 主源，行情和估值用可靠市场源交叉验证。
输出 buy-side memo：Serenity 瓶颈、财务、Bayesian growth、TAM-Adj-PEG、GF-DMA/缠论辅助、Bull/Base/Bear、证伪点。
请分开写：市场当前隐含的 H0-H5 增长，以及证据真正支持的 H0-H5 增长。
```

## 单股挑战

```text
用这个 Skill 挑战 [公司/代码] 的 thesis。
不要只找利好。请明确它的瓶颈属性、证据强度、财务兑现、估值透支程度和技术买点条件。
如果股价隐含 H4/H5，但证据只支持 H2/H3，请按证据等级降级。
最后给出：核心候选/强观察/高弹性主题/剔除。
```

## 数据审计

```text
对 [代码] 做数据审计。
解析市场和标准代码，列出需要的数据源，检查当前价、复权行情、财报、公告、股本、市值。
如果任何关键数据失败，请说明评级上限和不能得出的结论。
```
