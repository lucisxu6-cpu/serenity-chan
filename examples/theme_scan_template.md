# 主题扫描调用示例

```text
请用 serenity-chan-long-alpha 扫描 A 股机器人产业链。
目标：不是罗列概念股，而是找 1-3 个长线高胜率研究对象。
风险偏好：平衡偏进攻。
周期：1-3 年。

要求：
1. 先输出 Data Fetch Plan 和候选 universe 建立方法。
2. 先排产业链卡点层级：整机、执行器、丝杠、减速器、力传感、3D 视觉、电机、控制器、材料、加工。
3. 至少纳入 20 家 A 股/港股/未上市公司作为候选，再筛到 3-7 家。
4. 对 Top 候选给出证据等级、导入阶段、财务兑现、估值位置和缠论等待条件。
5. 明确哪些因为数据不足或证据弱被降级。
```
